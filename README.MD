# Personalized Content Dashboard

##  Project Overview
This is a Personalized Content Dashboard that displays dynamic news content using API integration. 
Users can search for news, view latest updates, and interact with the dashboard in a simple and responsive interface.

##  Features
- Fetch news from API
- Search functionality
- Responsive design
- Clean and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- News API

##  How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
